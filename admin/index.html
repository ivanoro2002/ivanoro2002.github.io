Qué parte de la web edita cada sección

01 Perfil y objetivo

Portada, resumen principal, datos rápidos, links personales y CV.

02 Proyectos

Nombre, objetivo, aprendizaje, próximo paso, tecnologías, imagen, demo y repositorio.

03 Cursos y habilidades

Agregar avances de aprendizaje, herramientas y próximos focos.

04 Apariencia y contacto

Cambiar el tema visual y actualizar Gmail, WhatsApp, LinkedIn, GitHub y CV.

Cómo subir o cambiar el CV

01 Abrir Perfil

Entrá a Editar contenido y abrí la sección Perfil.

02 Subir PDF

En Subir CV en PDF elegí el archivo desde tu computadora.

03 Guardar cambios

Guardá y publicá. El archivo queda dentro del sitio, no como ruta de tu PC.

04 Revisar web

En la vista pública, los botones de CV lo abren para ver o descargar.

Cómo cargar un proyecto

01 Crear repositorio

Subí el código a GitHub y copiá el link del repositorio.

02 Completar ficha

En el editor agregá nombre, estado, tipo, descripción y tecnologías.

03 Sumar evidencia

Pegá el link del repositorio y, si existe, el link de la demo publicada.

04 Imagen opcional

Subí una captura liviana si querés mostrar cómo se ve el proyecto.

Antes de publicar

  • Revisá que los enlaces abran: CV, GitHub, LinkedIn, Gmail y WhatsApp.
  • Si subís imágenes o CV, esperá a que el editor termine la carga antes de guardar.
  • Después de guardar, entrá a Ver web y confirmá que el cambio se vea bien.
