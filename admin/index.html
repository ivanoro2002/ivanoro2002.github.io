Que podes editar

01 Perfil y objetivo

Nombre, rol, resumen, datos rapidos y perfil de busqueda.

02 Proyectos

Nombre, objetivo, aprendizaje, proximo paso, tecnologias, imagen, demo y repositorio.

03 Cursos y habilidades

Agregar avances de aprendizaje, herramientas y proximos focos.

04 Apariencia y contacto

Cambiar el tema visual y actualizar Gmail, WhatsApp, LinkedIn, GitHub y CV.

Como cargar un proyecto

01 Crear repositorio

Subi el codigo a GitHub y copia el link del repositorio.

02 Completar ficha

En el editor agrega nombre, estado, tipo, descripcion y tecnologias.

03 Sumar evidencia

Pega el link del repositorio y, si existe, el link de la demo publicada.

04 Imagen opcional

Subi una captura liviana si queres mostrar como se ve el proyecto.
